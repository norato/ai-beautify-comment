Privacy Policy for AI Beautify Comment

Last updated: August 2025

Data Collection

AI Beautify Comment only processes:

  • Text content you select on web pages
  • Your API key (stored locally in your browser)

How We Use Data

Selected text is sent to Google's Gemini API for processing and enhancement. No data is stored on our servers.

Data Storage

All settings and API keys are stored locally in your browser using Chrome's storage API.
